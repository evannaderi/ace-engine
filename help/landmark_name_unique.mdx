### Multiple landmark roles of 'ia-haspopup` attribute that are disambiguated with a unique label or parent

### Why is this important?

PLACEHOLDER TEXT HERE SHOULD BE REPLACED BEFORE MERGE TO MASTER with `role="combobox"`, an `<element>` with values of `"listbox"`, `"tree"`, `"grid"`, or `"dialog"`.
If the value for the `'aria-haspopup'` attribute does not match the value for the popup element's `'role'` 
attribute, people who use assistive technologies may see unexpected behavior and will not know how 
to interact with the content.

<div id="locSnippet"></div>

### What to do

 * PLACEHOLDER TEXT HERE SHOULD BE REPLACED BEFORE MERGE TO MASTER Set the value of the `'aria-haspopup'` attribute of the `<combobox>` element to match the value of the `'role'` attribute of the implemented popup. . . .  Allowed `<popup>` element `'role'` attribute values are `"listbox"`, `"tree"`, `"grid"`, or `"dialog"`.
